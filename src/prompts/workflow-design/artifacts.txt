# Artifacts

- Route large generated content to files; prompts name exact output paths.
- Return `artifactPaths` plus a concise status or summary, not full file contents.
- Use `@workflow/...` with `readText` and `readJson` for workflow-owned prompts, schemas, fixtures, and small deterministic inputs. Use `writeText` and `writeJson` for artifacts and manifests that later stages read by path.
- Avoid bulk-reading project source just to paste it into a prompt; pass paths instead.
